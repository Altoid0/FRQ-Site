Linked Lists

A lab displaying the time complexity of linked lists vs. array lists.

Linked lists are a way to compile data into a list. Instead of setting aside space in memory during initialization like an array, linked lists are dynamic and do not utilize Java's Object class. Linked lists use primitive types and are thus more time efficient than array lists. They work by creating a series of nodes which point to each other. Said nodes can have their pointers changed in order to modify the list.
